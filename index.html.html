  • <!DOCTYPE html>: Declares the document type and version as HTML5, ensuring proper rendering by the browser.
  • <html>: The root element that encompasses the entire HTML document.
  • <head>: Contains meta-information like the title, character set, and links to scripts or styles.
  • <meta charset="UTF-8">: Sets the character encoding of the document to UTF-8, supporting all characters and symbols.
  • <title>: Defines the title of the HTML document, displayed in the browser's title bar or tab.
  • <body>: Contains all the visible content of the HTML document, like text, images, and links.
  • <h1>: Defines the main heading, typically used for the primary title of the document or section.
  • <h2>: Represents a second-level heading, used for major sections within the document.
  • <h3>: Represents a third-level heading, used for subsections within major sections.
  • <p>: Defines a paragraph, a block of text separated from other text by vertical space.
  • <a>: Defines a hyperlink, allowing users to navigate to another document or resource.
  • <ul>: Defines an unordered list, which is a bulleted list of items.
  • <li>: Defines a list item, used within <ul> or <ol> tags to represent individual list entries.
